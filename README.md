# ASAN KITAB (آسان کتاب) - Roman Urdu Quran

Asan Kitab ek aisi web application hai jo un logon ke liye banai gayi hai jinhen Urdu script parhne mein mushkil hoti hai. Is app ke zariye aap Quran-e-Pak ki har ayat ko uski **Arabic** text aur **Roman Urdu** translation ke saath parh sakte hain.

## Features:

- **Arabic & Roman Urdu**: Quran ki har ayat ke saath uska Roman Urdu tarjuma (Translation by Muhammad Junagarhi).
- **Chapter Selection**: Aap dropdown menu se ya Surah number likh kar koi bhi Surah khol sakte hain.
- **Save & Continue (Bookmark)**: Jab aap kisi Ayat ke number par click karte hain, to wo save ho jati hai. Jab aap dobara website kholenge, to app automatically aapko usi ayat par le jayegi jahan aapne chora tha.
- **Modern Design**: Premium dark-green theme (Islamic aesthetic) jo parhne mein sukoon deti hai.
- **Fast & Responsive**: Mobile aur computer dono par behtareen chalti hai.

## Kaise Chalayein?

Agar aap isay apne local computer par chalana chahte hain:

1. Sab se pehle project folder mein jayein.
2. Terminal ya Command Prompt kholein.
3. Niche di gayi command likhein:
   ```bash
   npm run dev
   ```
4. Phir apne browser mein `http://localhost:3000` khol kar parhna shuru karein.

## APIs Used:
- **Arabic**: Quran Uthmani Hafs (fawazahmed0 API)
- **Urdu Translation**: Muhammad Junagarhi (Roman Urdu)
